Build: Wi-Fi CC1310 Gateway - wifi1310

Sep 13, 2021 · 240 words · 2 minute read

Buildlog: wifi1310 🔗

The Goal 🔗

I wanted to make a small, user-friendly gateway. It should connect a sub-GHz 6lowpan mesh network (think eg Thread, or Amazon Sidewalk) to the Internet, via Wi-Fi.

The Challenge 🔗

Without the budget for custom RF work, nor space enough to be comfortable, make it work 😄.

Obviously RF must work and be compliant.

It must fit in a nice commercial off the shelf (COTS) pre-made enclosure, and provide mechanisms for USB power, status LEDs, button, programming and test points.

The Result 🔗

First revision had an improperly broken ground plane under the CC1310 module, so revision 2 it is. It caused spurious emissions from the sub-GHz module. Not great, not terrible.

Second revision worked pretty good!

Parts chosen:

  • Wi-Fi - Texas Instruments CC3220 module
  • Wi-Fi antenna - PCB antenna on the module
  • Sub-GHz - Texas Instruments CC1310 module, Radiocontrolli brand
  • Sub-GHz antenna - Yageo PCB antenna mounted via cable
  • Power - low-loss LDO from 5V via USB micro, to 3.3V for the system
  • LEDs - an edge-mounted, low-current, low-profile RGB LED from Cree
  • Enclosure - pre-made enclosure, available on Mouser

The PCB is an inexpensive 4-layer impedance controlled board made by JLPCB. Solder is applied by hand by stencil, and run through an soldering oven.
