Iceland: Aldeyjarfoss waterfall

Mar 31, 2023
Prio HÖG guidetoiceland Sneak peek 🔗What is it 🔗Otroligt vattenfall, kanske vackraste på Island. Sprunget ur basalt. Where is it 🔗Man kan med 4wd åka hela vägen fram, men med 2wd kan man ta sig ganska långt på 842an. Tar 40 minuter enkel väg från avfarten på R1, 37km.
